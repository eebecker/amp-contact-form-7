== Description ==

AMP Contact FORM 7 plugin will help you to enable contact form 7  in AMP.
You can use it with [AMP](https://wordpress.org/plugins/amp/) or [AMP for WP](https://wordpress.org/plugins/accelerated-mobile-pages/)  

[DEMO](https://woo.ciphercoin.com/demo-popup/?amp=1)


Extensions : [More contact form 7 add-ons](https://ciphercoin.com/contact-form-7-database-cfdb7-add-ons/) 
Support : [https://ciphercoin.com/contact/](https://ciphercoin.com/contact/)



== Installation ==

1. Download and extract plugin files to a wp-content/plugin directory.
2. Activate the plugin through the WordPress admin interface.
3. Done !
